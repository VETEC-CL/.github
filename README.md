# VETEC-CL

Repositorio inicial de la organización **VETEC-CL**.

## Descripción

Este espacio fue creado para centralizar el desarrollo, versionado y documentación de soluciones internas de la empresa.

## Tecnologías consideradas

- SQL
- Python
- PowerShell

## Objetivos

- mantener control de versiones
- ordenar los desarrollos por proyecto o cliente
- facilitar despliegues controlados a servidor
- reducir la dependencia de cambios manuales por escritorio remoto
- documentar procesos y configuraciones técnicas
